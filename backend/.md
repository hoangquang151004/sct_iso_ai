# Quy trình làm việc hệ thống SCT-ISO (Phiên bản Quản lý Quy tắc)

## 1. Thiết lập "Luật chơi" & Khung biểu mẫu
- **Vai trò:** ISO Manager  
- **Nền tảng:** Web  
- **Hành động chi tiết:**  
  - Cấu hình tài liệu, quy trình SOP.  
  - Tạo các mẫu Form giám sát (HACCP, PRP).  
  - Thiết lập các ngưỡng giới hạn tới hạn (Critical Limits) cho từng điểm kiểm soát.

---

## 2. Vận hành & Giám sát hiện trường
- **Vai trò:** User (Sản xuất) / QA/QC  
- **Nền tảng:** Web / Mobile  
- **Hành động chi tiết:**  
  - User: Tuân thủ làm việc theo SOP đã ban hành.  
  - QA/QC: Tải Form về điện thoại, kiểm tra xưởng, đo thông số, chụp ảnh minh chứng và gửi báo cáo.

---

## 3. Đối soát ngưỡng (Rule-based) & Khởi tạo lỗi (NC)
- **Vai trò:** Hệ thống tự động / QA Manager  
- **Nền tảng:** Tự động  
- **Hành động chi tiết:**  
  - Hệ thống tự động đối soát thông số nhập liệu với ngưỡng giới hạn (Critical Limits) đã cấu hình.  
  - Nếu dữ liệu vượt ngưỡng hoặc đánh giá "Không đạt", hệ thống tự động tạo Phiếu sự cố (Ticket NC) ở trạng thái Chờ xử lý.  
  - Gửi thông báo (Email/Notification) cho các bên liên quan.

---

## 4. Phân tích nguyên nhân & Đề xuất giải pháp
- **Vai trò:** User (Trưởng ca/Quản đốc bị lỗi)  
- **Nền tảng:** Web  
- **Hành động chi tiết:**  
  - Nhận thông báo Ticket NC.  
  - Viết giải trình: phân tích nguyên nhân gốc rễ (Root Cause) và kế hoạch khắc phục (Corrective Action Plan).  
  - Cập nhật vào đơn để đảm bảo hoàn thành đúng hạn.

---

## 5. Phê duyệt kế hoạch khắc phục
- **Vai trò:** ISO Manager / Trưởng phòng QA  
- **Nền tảng:** Web  
- **Hành động chi tiết:**  
  - Đọc kế hoạch giải quyết của User.  
  - Phê duyệt nếu giải pháp triệt để (User bắt đầu sửa chữa thực tế).  
  - Nếu chưa triệt để, gửi phản hồi yêu cầu viết lại.

---

## 6. Nghiệm thu & Đóng hồ sơ
- **Vai trò:** Auditor / ISO/QA Manager  
- **Nền tảng:** Mobile / Web  
- **Hành động chi tiết:**  
  - Khi hết hạn sửa chữa, hệ thống nhắc nhở kiểm tra lại (Verify).  
  - Auditor xuống hiện trường kiểm tra kết quả thực tế.  
  - Nếu lỗi đã khắc phục triệt để, ấn **Đóng hồ sơ / Close Ticket**.
